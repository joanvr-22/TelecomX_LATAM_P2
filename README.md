# TelecomX_LATAM_P2
challenge 3

Este proyecto corresponde a la segunda etapa del análisis de cancelación de clientes (churn) en TelecomX. El objetivo principal es desarrollar modelos predictivos capaces de estimar la probabilidad de que un cliente cancele su servicio a partir de variables relevantes relacionadas con sus características, servicios contratados y comportamiento de consumo. Para ello se utilizó el conjunto de datos previamente limpio y tratado en la primera parte del proyecto (TelecomX_LATAM_P1), el cual se carga en esta etapa como un archivo df_final.csv.

El proyecto se organiza principalmente en un cuaderno (notebook) donde se desarrolla todo el proceso de análisis y modelización. En el repositorio se incluye el notebook principal y el archivo de datos tratados df_final.csv generado en la primera etapa El flujo del cuaderno incluye la carga de los datos, preparación de variables, análisis exploratorio enfocado en churn, entrenamiento de modelos predictivos y evaluación de resultados.

En la preparación de los datos se clasificaron las variables en categóricas y numéricas. Algunos ejemplos de variables numéricas utilizadas fueron SeniorCitizen, tenure, Charges_Monthly, Charges_Total y cuentas_diarias. Por otro lado, entre las variables categóricas se incluyeron gender, Contract, InternetService y PaymentMethod. Las variables categóricas fueron codificadas para poder ser utilizadas por los modelos de machine learning, mientras que las variables numéricas fueron escaladas para asegurar que todas se encuentren en escalas comparables y evitar que algunas variables dominen el aprendizaje del modelo. Posteriormente se separaron las variables predictoras (X) de la variable objetivo (y), correspondiente a Churn, y se dividió el conjunto de datos en entrenamiento y prueba utilizando una proporción de 70% para entrenamiento y 30% para evaluación.

Durante el análisis exploratorio se generaron algunas visualizaciones para comprender mejor la relación entre ciertas variables y la cancelación de clientes. Entre ellas se construyó una matriz de correlación entre variables numéricas como SeniorCitizen, tenure, Charges_Monthly, Charges_Total y cuentas_diarias, incluyendo la variable Churn, con el objetivo de identificar posibles relaciones entre ellas. Además, se realizaron gráficos tipo boxplot para analizar la relación entre la cancelación y variables como el tiempo de permanencia del cliente (tenure) y el gasto total (Charges_Total), lo que permitió observar diferencias en la distribución de estas variables entre clientes que cancelan y aquellos que permanecen en la empresa.

Posteriormente se aplicaron distintos modelos de clasificación para predecir el churn, incluyendo regresión logística, random forest y support vector machine (SVM). Estos modelos permitieron evaluar qué variables influyen con mayor fuerza en la cancelación y comparar su desempeño utilizando métricas como accuracy, precisión, recall, F1-score y la matriz de confusión, así como analizar a los coeficientes de las variables, que muestran su contribución a la predicción de cancelación.

Para ejecutar el proyecto es necesario contar con Python y algunas bibliotecas de análisis de datos como pandas, numpy, scikit-learn, matplotlib y seaborn. Una vez instaladas estas dependencias, solo es necesario abrir el notebook, asegurarse de que el archivo df_final.csv se encuentre en el mismo directorio del proyecto y ejecutar las celdas en el orden en que aparecen para reproducir todo el proceso de preparación de datos, análisis exploratorio y entrenamiento de los modelos.
